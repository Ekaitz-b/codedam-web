    JavaFX - Interfaz Gráfica Moderna

    JavaFX es el framework moderno de Java para crear interfaces gráficas ricas y atractivas.

    Aplicación JavaFX Básica

    import javafx.application.Application; import javafx.scene.Scene; import javafx.scene.control.*; import javafx.scene.layout.*; import javafx.stage.Stage; public class MiAplicacion extends Application { @Override public void start(Stage primaryStage) { primaryStage.setTitle("CodeDAM App"); // Crear componentes Label etiqueta = new Label("Bienvenido a JavaFX"); TextField campoTexto = new TextField(); campoTexto.setPromptText("Introduce tu nombre"); Button boton = new Button("Saludar"); boton.setOnAction(e -> { String nombre = campoTexto.getText(); etiqueta.setText("Hola, " + nombre + "!"); }); // Layout VBox root = new VBox(10); root.getChildren().addAll(etiqueta, campoTexto, boton); root.setStyle("-fx-padding: 20; -fx-alignment: center;"); // Escena y mostrar Scene scene = new Scene(root, 400, 300); primaryStage.setScene(scene); primaryStage.show(); } public static void main(String[] args) { launch(args); } }

    Layouts JavaFX

    • VBox: Componentes en vertical
    • HBox: Componentes en horizontal
    • BorderPane: Top, Bottom, Left, Right, Center
    • GridPane: Cuadrícula de filas y columnas
    • StackPane: Componentes apilados
    • FlowPane: Flujo automático con wrap

    FXML - Diseño con XML

    <!-- vista.fxml --> <?xml version="1.0" encoding="UTF-8"?> <?import javafx.scene.control.*?> <?import javafx.scene.layout.*?> <VBox xmlns:fx="http://javafx.com/fxml" fx:controller="com.codedam.MiControlador" spacing="10" alignment="CENTER" style="-fx-padding: 20;"> <Label fx:id="etiqueta" text="Bienvenido"/> <TextField fx:id="campoNombre" promptText="Nombre"/> <Button text="Saludar" onAction="#handleSaludar"/> </VBox>
    // MiControlador.java import javafx.fxml.FXML; import javafx.scene.control.*; public class MiControlador { @FXML private Label etiqueta; @FXML private TextField campoNombre; @FXML private void handleSaludar() { String nombre = campoNombre.getText(); etiqueta.setText("Hola, " + nombre + "!"); } }

    CSS en JavaFX

    JavaFX permite estilizar interfaces con CSS, similar a desarrollo web.

    Estilos CSS JavaFX

    /* estilos.css */ .root { -fx-background-color: #f0f0f0; } .button { -fx-background-color: #667eea; -fx-text-fill: white; -fx-font-size: 14px; -fx-padding: 10 20 10 20; -fx-background-radius: 5; } .button:hover { -fx-background-color: #5568d3; -fx-cursor: hand; } .text-field { -fx-font-size: 14px; -fx-padding: 8; -fx-border-color: #ccc; -fx-border-radius: 4; } .label { -fx-font-size: 16px; -fx-font-weight: bold; -fx-text-fill: #333; }
    // Aplicar CSS scene.getStylesheets().add(getClass() .getResource("estilos.css").toExternalForm()); // Aplicar clase CSS a componente boton.getStyleClass().add("mi-boton-especial");

    Componentes JavaFX Avanzados

    TableView

    import javafx.collections.*; import javafx.scene.control.*; import javafx.scene.control.cell.PropertyValueFactory; public class Persona { private String nombre; private int edad; public Persona(String nombre, int edad) { this.nombre = nombre; this.edad = edad; } public String getNombre() { return nombre; } public int getEdad() { return edad; } } // Crear TableView TableView<Persona> tabla = new TableView<>(); TableColumn<Persona, String> colNombre = new TableColumn<>("Nombre"); colNombre.setCellValueFactory(new PropertyValueFactory<>("nombre")); TableColumn<Persona, Integer> colEdad = new TableColumn<>("Edad"); colEdad.setCellValueFactory(new PropertyValueFactory<>("edad")); tabla.getColumns().addAll(colNombre, colEdad); // Datos ObservableList<Persona> datos = FXCollections.observableArrayList( new Persona("Ana", 20), new Persona("Juan", 22), new Persona("María", 21) ); tabla.setItems(datos);

    Diseño UI/UX

    Principios de Diseño

    • Simplicidad: Interfaces limpias y fáciles de usar
    • Consistencia: Mantener patrones visuales
    • Feedback: Responder a acciones del usuario
    • Jerarquía Visual: Destacar elementos importantes
    • Accesibilidad: Usar able para todos los usuarios

    Colores y Tipografía

    • Usar esquemas de color coherentes
    • Contraste adecuado para legibilidad
    • Máximo 3 fuentes diferentes
    • Tamaños de fuente jerárquicos

    Espaciado y Alineación

    • Padding y margin consistentes
    • Alinear elementos correctamente
    • Usar grids para organización
    • Espacios en blanco (whitespace) apropiados

    Regla de los 3 clics: El usuario debe poder acceder a cualquier función en máximo 3 clics.

    Eventos y Manejo de Interacciones

    // Eventos de botón Button btn = new Button("Click"); btn.setOnAction(e -> System.out.println("Click!")); // Eventos de teclado textField.setOnKeyPressed(e -> { if (e.getCode() == KeyCode.ENTER) { System.out.println("Enter presionado"); } }); // Eventos de ratón label.setOnMouseEntered(e -> label.setStyle("-fx-text-fill: blue;")); label.setOnMouseExited(e -> label.setStyle("-fx-text-fill: black;")); // Listener de cambios slider.valueProperty().addListener((obs, oldVal, newVal) -> { System.out.println("Nuevo valor: " + newVal); });
