Introducción a UI/UX

El Desarrollo de Interfaces se centra en crear aplicaciones gráficas que sean funcionales, atractivas y fáciles de usar. Combina aspectos técnicos de programación con principios de diseño centrado en el usuario.

Diferencia entre UI y UX

  • UI (User Interface): Elementos visuales con los que interactúa el usuario (botones, menús, colores)
  • UX (User Experience): Experiencia global del usuario al usar la aplicación (facilidad, satisfacción)

Principios de Diseño de Interfaces

  • Simplicidad: Interfaces limpias sin elementos innecesarios
  • Consistencia: Mantener patrones visuales y de interacción uniformes
  • Feedback: Responder siempre a las acciones del usuario
  • Jerarquía Visual: Destacar elementos importantes mediante tamaño, color y posición
  • Affordance: Los elementos deben sugerir cómo usarlos
  • Prevención de Errores: Diseñar para evitar errores del usuario

Accesibilidad (a11y)

  • Contraste de colores adecuado (WCAG 2.1)
  • Textos alternativos en imágenes
  • Navegación por teclado
  • Tamaños de fuente legibles (mínimo 16px)
  • No depender solo del color para transmitir información

Regla de los 3 clics: El usuario debe poder acceder a cualquier función en máximo 3 clics.

JavaFX - Fundamentos

JavaFX es el framework moderno de Java para crear interfaces gráficas ricas. Reemplazó a Swing como la tecnología recomendada para aplicaciones de escritorio.

FXML y SceneBuilder

FXML permite separar la definición de la interfaz (XML) de la lógica (Java), siguiendo el patrón MVC. SceneBuilder es una herramienta visual para diseñar FXML.

Archivo FXML

<?xml version="1.0" encoding="UTF-8"?> <?import javafx.scene.control.*?> <?import javafx.scene.layout.*?> <?import javafx.geometry.Insets?> <BorderPane xmlns:fx="http://javafx.com/fxml" fx:controller="com.ejemplo.MainController" prefWidth="600" prefHeight="400"> <top> <HBox alignment="CENTER" style="-fx-background-color: #667eea; -fx-padding: 15;"> <Label text="Gestión de Productos" style="-fx-text-fill: white; -fx-font-size: 18px;"/> </HBox> </top> <center> <VBox spacing="15" alignment="CENTER"> <padding><Insets top="20" right="20" bottom="20" left="20"/></padding> <GridPane hgap="10" vgap="10"> <Label text="Nombre:" GridPane.columnIndex="0" GridPane.rowIndex="0"/> <TextField fx:id="txtNombre" GridPane.columnIndex="1" GridPane.rowIndex="0"/> <Label text="Precio:" GridPane.columnIndex="0" GridPane.rowIndex="1"/> <TextField fx:id="txtPrecio" GridPane.columnIndex="1" GridPane.rowIndex="1"/> <Label text="Categoría:" GridPane.columnIndex="0" GridPane.rowIndex="2"/> <ComboBox fx:id="cboCategoria" GridPane.columnIndex="1" GridPane.rowIndex="2"/> </GridPane> <HBox spacing="10" alignment="CENTER"> <Button text="Guardar" onAction="#handleGuardar"/> <Button text="Limpiar" onAction="#handleLimpiar"/> </HBox> <Label fx:id="lblMensaje" style="-fx-text-fill: green;"/> </VBox> </center> </BorderPane>

Controlador Java

package com.ejemplo; import javafx.fxml.FXML; import javafx.fxml.Initializable; import javafx.scene.control.*; import java.net.URL; import java.util.ResourceBundle; public class MainController implements Initializable { @FXML private TextField txtNombre; @FXML private TextField txtPrecio; @FXML private ComboBox<String> cboCategoria; @FXML private Label lblMensaje; @Override public void initialize(URL url, ResourceBundle rb) { // Inicializar componentes cboCategoria.getItems().addAll( "Electrónica", "Ropa", "Hogar", "Deportes" ); cboCategoria.setValue("Electrónica"); } @FXML private void handleGuardar() { String nombre = txtNombre.getText(); String precio = txtPrecio.getText(); String categoria = cboCategoria.getValue(); if (nombre.isEmpty() || precio.isEmpty()) { lblMensaje.setText("Complete todos los campos"); lblMensaje.setStyle("-fx-text-fill: red;"); return; } try { double precioNum = Double.parseDouble(precio); // Guardar producto... lblMensaje.setText("Producto guardado: " + nombre); lblMensaje.setStyle("-fx-text-fill: green;"); handleLimpiar(); } catch (NumberFormatException e) { lblMensaje.setText("Precio inválido"); lblMensaje.setStyle("-fx-text-fill: red;"); } } @FXML private void handleLimpiar() { txtNombre.clear(); txtPrecio.clear(); cboCategoria.setValue("Electrónica"); } }

public class FormularioRegistro extends Application { @Override public void start(Stage stage) { GridPane grid = new GridPane(); grid.setHgap(10); grid.setVgap(10); grid.setPadding(new Insets(25)); // Campos TextField txtUsuario = new TextField(); PasswordField txtPassword = new PasswordField(); PasswordField txtConfirmar = new PasswordField(); TextField txtEmail = new TextField(); Label lblError = new Label(); lblError.setStyle("-fx-text-fill: red;"); // Layout grid.add(new Label("Usuario:"), 0, 0); grid.add(txtUsuario, 1, 0); grid.add(new Label("Email:"), 0, 1); grid.add(txtEmail, 1, 1); grid.add(new Label("Contraseña:"), 0, 2); grid.add(txtPassword, 1, 2); grid.add(new Label("Confirmar:"), 0, 3); grid.add(txtConfirmar, 1, 3); Button btnRegistrar = new Button("Registrar"); grid.add(btnRegistrar, 1, 4); grid.add(lblError, 0, 5, 2, 1); // Validación btnRegistrar.setOnAction(e -> { String usuario = txtUsuario.getText(); String email = txtEmail.getText(); String pass = txtPassword.getText(); String confirm = txtConfirmar.getText(); if (usuario.isEmpty() || email.isEmpty() || pass.isEmpty()) { lblError.setText("Todos los campos son obligatorios"); } else if (!email.matches("^[\\w.-]+@[\\w.-]+\\.[a-z]{2,}$")) { lblError.setText("Email inválido"); } else if (pass.length() < 6) { lblError.setText("Contraseña mínimo 6 caracteres"); } else if (!pass.equals(confirm)) { lblError.setText("Las contraseñas no coinciden"); } else { lblError.setStyle("-fx-text-fill: green;"); lblError.setText("¡Registro exitoso!"); } }); Scene scene = new Scene(grid, 350, 250); stage.setTitle("Registro"); stage.setScene(scene); stage.show(); } }

Ejercicio 2: Calculadora

Calculadora básica con operaciones aritméticas.

public class Calculadora extends Application { private TextField display; private double operando1 = 0; private String operacion = ""; @Override public void start(Stage stage) { VBox root = new VBox(10); root.setPadding(new Insets(10)); display = new TextField("0"); display.setEditable(false); display.setAlignment(Pos.CENTER_RIGHT); display.setStyle("-fx-font-size: 24px;"); GridPane botones = new GridPane(); botones.setHgap(5); botones.setVgap(5); String[][] teclas = { {"7", "8", "9", "/"}, {"4", "5", "6", "*"}, {"1", "2", "3", "-"}, {"C", "0", "=", "+"} }; for (int i = 0; i < 4; i++) { for (int j = 0; j < 4; j++) { Button btn = new Button(teclas[i][j]); btn.setPrefSize(50, 50); btn.setOnAction(this::handleClick); botones.add(btn, j, i); } } root.getChildren().addAll(display, botones); Scene scene = new Scene(root); stage.setTitle("Calculadora"); stage.setScene(scene); stage.show(); } private void handleClick(ActionEvent e) { String texto = ((Button) e.getSource()).getText(); switch (texto) { case "C": display.setText("0"); operando1 = 0; operacion = ""; break; case "+": case "-": case "*": case "/": operando1 = Double.parseDouble(display.getText()); operacion = texto; display.setText("0"); break; case "=": double operando2 = Double.parseDouble(display.getText()); double resultado = calcular(operando1, operando2, operacion); display.setText(String.valueOf(resultado)); break; default: if (display.getText().equals("0")) { display.setText(texto); } else { display.setText(display.getText() + texto); } } } private double calcular(double a, double b, String op) { return switch (op) { case "+" -> a + b; case "-" -> a - b; case "*" -> a * b; case "/" -> b != 0 ? a / b : 0; default -> b; }; } }

Ejercicio 3: Lista de Tareas

Aplicación para gestionar tareas pendientes.

public class TodoApp extends Application { private ObservableList<String> tareas; private ListView<String> listView; @Override public void start(Stage stage) { VBox root = new VBox(10); root.setPadding(new Insets(15)); // Input HBox inputBox = new HBox(10); TextField txtTarea = new TextField(); txtTarea.setPromptText("Nueva tarea..."); HBox.setHgrow(txtTarea, Priority.ALWAYS); Button btnAgregar = new Button("Agregar"); inputBox.getChildren().addAll(txtTarea, btnAgregar); // Lista tareas = FXCollections.observableArrayList(); listView = new ListView<>(tareas); VBox.setVgrow(listView, Priority.ALWAYS); // Botones de acción HBox actionBox = new HBox(10); Button btnEliminar = new Button("Eliminar"); Button btnLimpiar = new Button("Limpiar todo"); actionBox.getChildren().addAll(btnEliminar, btnLimpiar); // Eventos btnAgregar.setOnAction(e -> { String tarea = txtTarea.getText().trim(); if (!tarea.isEmpty()) { tareas.add(tarea); txtTarea.clear(); } }); txtTarea.setOnKeyPressed(e -> { if (e.getCode() == KeyCode.ENTER) { btnAgregar.fire(); } }); btnEliminar.setOnAction(e -> { int index = listView.getSelectionModel().getSelectedIndex(); if (index >= 0) { tareas.remove(index); } }); btnLimpiar.setOnAction(e -> tareas.clear()); root.getChildren().addAll( new Label("Lista de Tareas"), inputBox, listView, actionBox ); Scene scene = new Scene(root, 400, 500); stage.setTitle("Todo App"); stage.setScene(scene); stage.show(); } }

Ejercicio 4: Conversor de Unidades

Conversor de temperatura, longitud y peso.

public class Conversor extends Application { @Override public void start(Stage stage) { TabPane tabPane = new TabPane(); tabPane.getTabs().addAll( crearTabTemperatura(), crearTabLongitud(), crearTabPeso() ); Scene scene = new Scene(tabPane, 400, 300); stage.setTitle("Conversor de Unidades"); stage.setScene(scene); stage.show(); } private Tab crearTabTemperatura() { Tab tab = new Tab("Temperatura"); tab.setClosable(false); VBox content = new VBox(15); content.setPadding(new Insets(20)); content.setAlignment(Pos.CENTER); TextField txtValor = new TextField(); txtValor.setPromptText("Valor"); txtValor.setMaxWidth(150); ComboBox<String> cboOrigen = new ComboBox<>(); cboOrigen.getItems().addAll("Celsius", "Fahrenheit", "Kelvin"); cboOrigen.setValue("Celsius"); ComboBox<String> cboDestino = new ComboBox<>(); cboDestino.getItems().addAll("Celsius", "Fahrenheit", "Kelvin"); cboDestino.setValue("Fahrenheit"); Label lblResultado = new Label("Resultado: "); lblResultado.setStyle("-fx-font-size: 16px; -fx-font-weight: bold;"); Button btnConvertir = new Button("Convertir"); btnConvertir.setOnAction(e -> { try { double valor = Double.parseDouble(txtValor.getText()); double resultado = convertirTemp(valor, cboOrigen.getValue(), cboDestino.getValue()); lblResultado.setText(String.format("Resultado: %.2f %s", resultado, cboDestino.getValue())); } catch (NumberFormatException ex) { lblResultado.setText("Valor inválido"); } }); content.getChildren().addAll( new HBox(10, new Label("De:"), cboOrigen), txtValor, new HBox(10, new Label("A:"), cboDestino), btnConvertir, lblResultado ); tab.setContent(content); return tab; } private double convertirTemp(double valor, String origen, String destino) { // Convertir a Celsius primero double celsius = switch (origen) { case "Fahrenheit" -> (valor - 32) * 5 / 9; case "Kelvin" -> valor - 273.15; default -> valor; }; // Convertir de Celsius al destino return switch (destino) { case "Fahrenheit" -> celsius * 9 / 5 + 32; case "Kelvin" -> celsius + 273.15; default -> celsius; }; } // Métodos similares para crearTabLongitud() y crearTabPeso() }

Ejercicio 5: Visor de Imágenes

Galería de imágenes con navegación.

public class VisorImagenes extends Application { private ImageView imageView; private List<String> imagenes; private int indiceActual = 0; @Override public void start(Stage stage) { BorderPane root = new BorderPane(); // Cargar imágenes imagenes = List.of( "/img/foto1.jpg", "/img/foto2.jpg", "/img/foto3.jpg" ); // ImageView imageView = new ImageView(); imageView.setPreserveRatio(true); imageView.setFitWidth(500); StackPane center = new StackPane(imageView); center.setStyle("-fx-background-color: #333;"); root.setCenter(center); // Navegación Button btnAnterior = new Button("◀"); btnAnterior.setOnAction(e -> mostrarImagen(-1)); Button btnSiguiente = new Button("▶"); btnSiguiente.setOnAction(e -> mostrarImagen(1)); HBox nav = new HBox(20, btnAnterior, btnSiguiente); nav.setAlignment(Pos.CENTER); nav.setPadding(new Insets(10)); root.setBottom(nav); // Mostrar primera imagen cargarImagen(imagenes.get(0)); // Navegación con teclado root.setOnKeyPressed(e -> { switch (e.getCode()) { case LEFT -> mostrarImagen(-1); case RIGHT -> mostrarImagen(1); } }); Scene scene = new Scene(root, 600, 500); stage.setTitle("Visor de Imágenes"); stage.setScene(scene); stage.show(); root.requestFocus(); } private void mostrarImagen(int direccion) { indiceActual = (indiceActual + direccion + imagenes.size()) % imagenes.size(); cargarImagen(imagenes.get(indiceActual)); } private void cargarImagen(String ruta) { try { Image img = new Image(getClass().getResourceAsStream(ruta)); imageView.setImage(img); } catch (Exception e) { System.err.println("Error cargando imagen: " + ruta); } } }

Ejercicio 6: Editor de Texto Simple

Editor con menú Archivo (Nuevo, Abrir, Guardar).

public class EditorTexto extends Application { private TextArea textArea; private Stage stage; private File archivoActual; @Override public void start(Stage primaryStage) { this.stage = primaryStage; BorderPane root = new BorderPane(); // Menú MenuBar menuBar = new MenuBar(); Menu menuArchivo = new Menu("Archivo"); MenuItem itemNuevo = new MenuItem("Nuevo"); itemNuevo.setAccelerator(KeyCombination.keyCombination("Ctrl+N")); itemNuevo.setOnAction(e -> nuevo()); MenuItem itemAbrir = new MenuItem("Abrir"); itemAbrir.setAccelerator(KeyCombination.keyCombination("Ctrl+O")); itemAbrir.setOnAction(e -> abrir()); MenuItem itemGuardar = new MenuItem("Guardar"); itemGuardar.setAccelerator(KeyCombination.keyCombination("Ctrl+S")); itemGuardar.setOnAction(e -> guardar()); MenuItem itemSalir = new MenuItem("Salir"); itemSalir.setOnAction(e -> Platform.exit()); menuArchivo.getItems().addAll(itemNuevo, itemAbrir, itemGuardar, new SeparatorMenuItem(), itemSalir); menuBar.getMenus().add(menuArchivo); root.setTop(menuBar); // Área de texto textArea = new TextArea(); textArea.setWrapText(true); root.setCenter(textArea); Scene scene = new Scene(root, 600, 400); stage.setTitle("Editor de Texto"); stage.setScene(scene); stage.show(); } private void nuevo() { textArea.clear(); archivoActual = null; stage.setTitle("Editor de Texto - Nuevo"); } private void abrir() { FileChooser fileChooser = new FileChooser(); fileChooser.setTitle("Abrir archivo"); fileChooser.getExtensionFilters().add( new FileChooser.ExtensionFilter("Archivos de texto", "*.txt") ); File file = fileChooser.showOpenDialog(stage); if (file != null) { try { String contenido = Files.readString(file.toPath()); textArea.setText(contenido); archivoActual = file; stage.setTitle("Editor - " + file.getName()); } catch (IOException e) { mostrarError("Error al abrir archivo"); } } } private void guardar() { if (archivoActual == null) { FileChooser fileChooser = new FileChooser(); fileChooser.setTitle("Guardar archivo"); archivoActual = fileChooser.showSaveDialog(stage); } if (archivoActual != null) { try { Files.writeString(archivoActual.toPath(), textArea.getText()); stage.setTitle("Editor - " + archivoActual.getName()); } catch (IOException e) { mostrarError("Error al guardar"); } } } private void mostrarError(String mensaje) { Alert alert = new Alert(Alert.AlertType.ERROR); alert.setContentText(mensaje); alert.showAndWait(); } }

Ejercicio 8: Dashboard con Gráficos

import javafx.scene.chart.*; public class Dashboard extends Application { @Override public void start(Stage stage) { BorderPane root = new BorderPane(); // Título Label titulo = new Label("Dashboard de Ventas"); titulo.setStyle("-fx-font-size: 24px; -fx-font-weight: bold;"); BorderPane.setAlignment(titulo, Pos.CENTER); BorderPane.setMargin(titulo, new Insets(20)); root.setTop(titulo); // Gráficos en grid GridPane charts = new GridPane(); charts.setHgap(20); charts.setVgap(20); charts.setPadding(new Insets(20)); // Gráfico de barras CategoryAxis xAxis = new CategoryAxis(); NumberAxis yAxis = new NumberAxis(); BarChart<String, Number> barChart = new BarChart<>(xAxis, yAxis); barChart.setTitle("Ventas por Mes"); XYChart.Series<String, Number> series = new XYChart.Series<>(); series.setName("2024"); series.getData().addAll( new XYChart.Data<>("Ene", 150), new XYChart.Data<>("Feb", 200), new XYChart.Data<>("Mar", 180), new XYChart.Data<>("Abr", 250) ); barChart.getData().add(series); // Gráfico circular PieChart pieChart = new PieChart(); pieChart.setTitle("Ventas por Categoría"); pieChart.getData().addAll( new PieChart.Data("Electrónica", 35), new PieChart.Data("Ropa", 25), new PieChart.Data("Hogar", 20), new PieChart.Data("Otros", 20) ); // Gráfico de líneas NumberAxis xAxis2 = new NumberAxis(); NumberAxis yAxis2 = new NumberAxis(); LineChart<Number, Number> lineChart = new LineChart<>(xAxis2, yAxis2); lineChart.setTitle("Tendencia"); XYChart.Series<Number, Number> lineSeries = new XYChart.Series<>(); lineSeries.setName("Ventas"); for (int i = 1; i <= 12; i++) { lineSeries.getData().add( new XYChart.Data<>(i, Math.random() * 100 + 100) ); } lineChart.getData().add(lineSeries); charts.add(barChart, 0, 0); charts.add(pieChart, 1, 0); charts.add(lineChart, 0, 1, 2, 1); root.setCenter(charts); Scene scene = new Scene(root, 900, 700); stage.setTitle("Dashboard"); stage.setScene(scene); stage.show(); } }
