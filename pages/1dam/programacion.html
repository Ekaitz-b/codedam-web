    Introducción a Java

    Java es un lenguaje de programación orientado a objetos, robusto, seguro y multiplataforma. Es uno de los lenguajes más utilizados en el mundo del desarrollo de software empresarial y aplicaciones móviles Android.

    Características de Java

    • Multiplataforma: "Write Once, Run Anywhere" - compilado a bytecode que se ejecuta en la JVM
    • Orientado a Objetos: Todo en Java es un objeto (excepto tipos primitivos)
    • Robusto: Gestión automática de memoria (Garbage Collector)
    • Seguro: No usa punteros, sistema de seguridad incorporado
    • Distribuido: Facilita la creación de aplicaciones distribuidas

    Sintaxis Básica

    public class HolaMundo { public static void main(String[] args) { System.out.println("Hola, CodeDAM!"); // Variables int numero = 42; String texto = "Java"; double decimal = 3.14; boolean verdadero = true; } }

    Programación Orientada a Objetos (POO)

    Clases y Objetos

    Una clase es una plantilla que define las características y comportamientos de un objeto. Un objeto es una instancia de una clase.

    public class Estudiante { // Atributos (características) private String nombre; private int edad; private double notaMedia; // Constructor public Estudiante(String nombre, int edad) { this.nombre = nombre; this.edad = edad; this.notaMedia = 0.0; } // Métodos (comportamientos) public void estudiar() { System.out.println(nombre + " está estudiando..."); } public void setNotaMedia(double nota) { if (nota >= 0 && nota <= 10) { this.notaMedia = nota; } } public double getNotaMedia() { return this.notaMedia; } } // Uso: Estudiante alumno = new Estudiante("Ana", 20); alumno.estudiar(); alumno.setNotaMedia(8.5);

    Pilares de la POO

    1. Encapsulación

    Ocultar los detalles internos de una clase y exponer solo lo necesario mediante modificadores de acceso.

    • private: Solo accesible dentro de la clase
    • public: Accesible desde cualquier parte
    • protected: Accesible en la clase y sus subclases
    • default: Accesible en el mismo paquete

    2. Herencia

    Permite crear clases derivadas que heredan características de una clase base.

    public class Persona { protected String nombre; protected int edad; public void saludar() { System.out.println("Hola, soy " + nombre); } } public class Profesor extends Persona { private String asignatura; public void enseñar() { System.out.println(nombre + " enseña " + asignatura); } } // Uso: Profesor prof = new Profesor(); prof.nombre = "Carlos"; prof.saludar(); // Heredado de Persona prof.enseñar(); // Propio de Profesor

    3. Polimorfismo

    Capacidad de un objeto de tomar muchas formas. Permite que diferentes clases respondan de manera distinta al mismo método.

    public abstract class Animal { public abstract void hacerSonido(); } public class Perro extends Animal { @Override public void hacerSonido() { System.out.println("Guau guau!"); } } public class Gato extends Animal { @Override public void hacerSonido() { System.out.println("Miau!"); } } // Polimorfismo en acción: Animal miMascota = new Perro(); miMascota.hacerSonido(); // "Guau guau!" miMascota = new Gato(); miMascota.hacerSonido(); // "Miau!"

    4. Abstracción

    Ocultar la complejidad mostrando solo la funcionalidad esencial.

    public interface Vehiculo { void acelerar(); void frenar(); void girar(String direccion); } public class Coche implements Vehiculo { @Override public void acelerar() { System.out.println("El coche acelera..."); } @Override public void frenar() { System.out.println("El coche frena..."); } @Override public void girar(String direccion) { System.out.println("Girando a " + direccion); } }

    Estructuras de Datos

    Arrays

    Estructura de datos que almacena elementos del mismo tipo en posiciones consecutivas de memoria.

    // Declaración e inicialización int[] numeros = new int[5]; String[] nombres = {"Ana", "Juan", "María"}; // Acceso a elementos numeros[0] = 10; System.out.println(nombres[1]); // "Juan" // Recorrer array for (int i = 0; i < numeros.length; i++) { System.out.println(numeros[i]); } // For-each for (String nombre : nombres) { System.out.println(nombre); }

    ArrayList

    Lista dinámica que puede crecer y decrecer en tamaño.

    import java.util.ArrayList; ArrayList<String> lista = new ArrayList<>(); // Agregar elementos lista.add("Java"); lista.add("Python"); lista.add("JavaScript"); // Acceder a elementos String primero = lista.get(0); // Modificar elementos lista.set(1, "C++"); // Eliminar elementos lista.remove(2); lista.remove("Java"); // Tamaño int tamaño = lista.size(); // Verificar si contiene boolean existe = lista.contains("Python");

    HashMap

    Estructura clave-valor para almacenar y recuperar datos eficientemente.

    import java.util.HashMap; HashMap<String, Integer> edades = new HashMap<>(); // Agregar pares clave-valor edades.put("Ana", 20); edades.put("Juan", 22); edades.put("María", 21); // Obtener valor int edadAna = edades.get("Ana"); // Verificar si existe clave if (edades.containsKey("Juan")) { System.out.println("Juan está en el mapa"); } // Recorrer HashMap for (String nombre : edades.keySet()) { System.out.println(nombre + ": " + edades.get(nombre)); }

    Manejo de Excepciones

    Las excepciones son eventos que interrumpen el flujo normal de ejecución. Java proporciona un mecanismo robusto para manejarlas.

    Try-Catch

    try { int resultado = 10 / 0; // ArithmeticException } catch (ArithmeticException e) { System.out.println("Error: División por cero"); e.printStackTrace(); } finally { // Se ejecuta siempre System.out.println("Bloque finally"); } // Múltiples catch try { String texto = null; System.out.println(texto.length()); } catch (NullPointerException e) { System.out.println("Error: Objeto null"); } catch (Exception e) { System.out.println("Error genérico"); }

    Lanzar Excepciones

    public void validarEdad(int edad) throws Exception { if (edad < 0) { throw new Exception("La edad no puede ser negativa"); } if (edad < 18) { throw new Exception("Debe ser mayor de edad"); } } // Crear excepción personalizada public class EdadInvalidaException extends Exception { public EdadInvalidaException(String mensaje) { super(mensaje); } }

    Consejos para Aprender Java

    Práctica constante: La programación se aprende programando. Practica con ejercicios diarios y proyectos pequeños.

    • Empieza con proyectos simples (calculadora, gestor de tareas)
    • Lee código de otros desarrolladores en GitHub
    • Participa en desafíos de programación (HackerRank, LeetCode)
    • Documenta tu código desde el principio
    • Aprende a usar un IDE (IntelliJ IDEA, Eclipse, NetBeans)
    • Estudia las convenciones de nomenclatura de Java

    Errores comunes: No gestionar excepciones, no cerrar recursos (archivos, conexiones), usar == para comparar Strings (usa .equals()).
