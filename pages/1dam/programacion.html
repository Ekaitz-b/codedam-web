    Errores Comunes con M√©todos:

    • Tipo de retorno incorrecto: public static int metodo() { return "texto"; } ‚Üí ERROR
    • Llamar m√©todo no-static desde static: No puedes llamar m√©todos de instancia desde main sin crear un objeto
    • Recursi√≥n sin caso base: Provoca StackOverflowError
    • Modificar par√°metros primitivos: Los cambios NO afectan al original

    Trucos Pro - M√©todos:

    • Nombra m√©todos con verbos: calcular, obtener, verificar, imprimir
    • Un m√©todo debe hacer UNA cosa y hacerla bien (Single Responsibility Principle)
    • M√°ximo 3-4 par√°metros; si necesitas m√°s, considera usar un objeto
    • M√©todos cortos (m√°ximo 20-30 l√≠neas) son m√°s legibles y mantenibles
    • Documenta m√©todos complejos con comentarios o JavaDoc
