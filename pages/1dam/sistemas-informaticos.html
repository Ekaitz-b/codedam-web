Linux, Windows, Redes y Arquitectura de Computadores

Índice de Contenidos

    Arquitectura de Computadores

    Componentes Principales

    • CPU (Procesador): Cerebro del ordenador, ejecuta instrucciones
    • RAM (Memoria): Almacenamiento temporal de datos en uso
    • Disco Duro/SSD: Almacenamiento permanente
    • Placa Base: Conecta todos los componentes
    • Fuente de Alimentación: Suministra energía
    • GPU (Tarjeta Gráfica): Procesamiento de gráficos

    Arquitectura von Neumann

    Modelo básico de arquitectura de computadoras:

    • Unidad Central de Procesamiento (CPU)
    • Memoria (almacena datos e instrucciones)
    • Dispositivos de Entrada/Salida
    • Bus de datos (conexión entre componentes)

    Sistemas Operativos Linux

    Linux es fundamental para desarrolladores. Sistema operativo de código abierto basado en Unix.

    Distribuciones Populares

    • Ubuntu: Más popular, fácil para principiantes
    • Debian: Estable, base de Ubuntu
    • Fedora: Tecnologías de vanguardia
    • CentOS/Rocky Linux: Para servidores empresariales
    • Arch Linux: Personalizable, para usuarios avanzados

    Comandos Básicos Linux

    # Navegación pwd # Directorio actual ls # Listar archivos ls -la # Listar con detalles y ocultos cd /home/usuario # Cambiar directorio cd .. # Directorio padre # Archivos y Directorios mkdir carpeta # Crear directorio touch archivo.txt # Crear archivo vacío cp origen destino # Copiar mv origen destino # Mover/Renombrar rm archivo.txt # Eliminar archivo rm -r carpeta # Eliminar directorio recursivo cat archivo.txt # Ver contenido nano archivo.txt # Editar con nano vim archivo.txt # Editar con vim # Permisos chmod 755 script.sh # Cambiar permisos chmod +x script.sh # Hacer ejecutable chown usuario archivo # Cambiar propietario ls -l # Ver permisos # Procesos ps aux # Ver procesos top # Monitor de procesos htop # Monitor mejorado kill PID # Matar proceso killall nombre # Matar por nombre # Sistema df -h # Espacio en disco free -h # Memoria RAM uname -a # Info del sistema whoami # Usuario actual sudo comando # Ejecutar como admin

    Gestión de Paquetes

    # Ubuntu/Debian (APT) sudo apt update # Actualizar lista sudo apt upgrade # Actualizar paquetes sudo apt install paquete # Instalar sudo apt remove paquete # Desinstalar sudo apt search nombre # Buscar # Fedora/CentOS (DNF/YUM) sudo dnf install paquete sudo dnf update # Arch Linux (Pacman) sudo pacman -S paquete sudo pacman -Syu # Actualizar sistema

    Redes de Computadores

    Modelo OSI (7 Capas)

    1. Física: Transmisión de bits por medios físicos
    2. Enlace: Comunicación entre nodos adyacentes (MAC)
    3. Red: Enrutamiento de paquetes (IP)
    4. Transporte: Transmisión extremo a extremo (TCP/UDP)
    5. Sesión: Gestión de sesiones
    6. Presentación: Formato de datos
    7. Aplicación: Servicios de red (HTTP, FTP, SMTP)

    Modelo TCP/IP (4 Capas)

    • Acceso a Red: Hardware y enlace
    • Internet: Direccionamiento IP
    • Transporte: TCP y UDP
    • Aplicación: Protocolos de alto nivel

    Direcciones IP

    # IPv4 (32 bits) 192.168.1.100 # IP privada 10.0.0.1 # IP privada 8.8.8.8 # IP pública (Google DNS) # Máscara de subred 255.255.255.0 # /24 (256 IPs) 255.255.0.0 # /16 (65,536 IPs) # IPv6 (128 bits) 2001:0db8:85a3:0000:0000:8a2e:0370:7334 # Comandos de red ip addr # Ver IPs (Linux) ipconfig # Ver IPs (Windows) ping google.com # Probar conectividad traceroute google.com # Ruta de paquetes netstat -tuln # Puertos abiertos nslookup google.com # Resolver DNS curl ifconfig.me # Ver IP pública

    Protocolos Importantes

    • HTTP/HTTPS: Navegación web (puerto 80/443)
    • FTP: Transferencia de archivos (puerto 21)
    • SSH: Conexión segura remota (puerto 22)
    • SMTP: Envío de correo (puerto 25)
    • DNS: Resolución de nombres (puerto 53)
    • DHCP: Asignación automática de IPs

    • Administrador de Tareas: Ctrl+Shift+Esc
    • Servicios: services.msc
    • Registro: regedit
    • Eventos: eventvwr
    • Usuarios: lusrmgr.msc

    Virtualización

    Tecnologías de Virtualización

    • VirtualBox: Gratuito, fácil de usar
    • VMware: Profesional, muy potente
    • Hyper-V: Integrado en Windows Pro
    • Docker: Contenedores ligeros
    • KVM: Virtualización en Linux

    Conceptos de Virtualización

    • Hipervisor: Software que gestiona máquinas virtuales
    • VM (Máquina Virtual): Sistema operativo virtualizado
    • Snapshot: Captura del estado de la VM
    • Contenedor: Virtualización a nivel de aplicación

    Ventajas de Virtualización: Ahorro de costes, facilidad de pruebas, aislamiento, recuperación rápida.

    Seguridad Básica

    Buenas Prácticas

    • Usar contraseñas fuertes y únicas
    • Mantener sistema y software actualizados
    • Usar firewall activado
    • Antivirus/antimalware actualizado
    • Hacer backups regularmente
    • No ejecutar software de fuentes desconocidas
    • Usar VPN en redes públicas
    • Autenticación de dos factores (2FA)

    Tipos de Amenazas

    • Malware: Software malicioso (virus, troyanos)
    • Phishing: Suplantación de identidad
    • Ransomware: Secuestro de datos
    • DDoS: Ataque de denegación de servicio
    • Man in the Middle: Interceptación de comunicaciones
